Proyectos

Norris

Norris surgió como un proyecto educativo, enfocado en ser un asistente virtual especializado en brindar apoyo a pacientes con EPOC.Este no solo proporciona un sistema para el registro de síntomas y signos vitales, sino que también facilita la comunicación directa entre médicos y pacientes a través de un chat que incluye un historial de conversaciones. Además, genera estadísticas y cuenta con un blog informativo sobre la enfermedad. Norris incorpora un asistente virtual inteligente, diseñado específicamente para pacientes con EPOC, y ha sido desarrollado utilizando tecnologías como Laravel, MySQL, MongoDB, CSS, JavaScript, PHP, HTML, Bootstrap, Vue, y la API de OpenAI. Dentro del proyecto, mi rol consistió en el testing y el desarrollo de front-end.

EscolarBite

EscolarBite surge como una iniciativa educativa diseñada para transformar la forma en que los estudiantes de la Universidad Gerardo Barrios (UGB) ordenan alimentos. A través de una aplicación web, se proporciona una solución digital innovadora que optimiza la experiencia de pedido en los cafetines. Con una interfaz amigable e intuitiva que permite a los usuarios explorar diversas opciones de menús, personalizar sus pedidos de acuerdo con sus preferencias y realizar solicitudes en línea. Esto no solo hace más eficiente el proceso de pedido de alimentos, sino que también ofrece a los estudiantes la posibilidad de monitorear el tiempo estimado para la preparación de sus órdenes y recibir notificaciones cuando estén listas para ser recogidas. En este proyecto, desempeñé el rol de desarrollador front-end, contribuyendo a la creación de una experiencia de usuario atractiva y funcional. Para el desarrollo se usaron tecnologías como PHP, HTML5, CSS, Bootstrap JS y SweetAlert2.
